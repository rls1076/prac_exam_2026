## Setup Instructions (get help as needed): 
1. Fork my 'prac_exam_2026' repo
2. Open vscode. Go to 'file --> new window'. Close all other instances of vscode
3. Hit 'cmd + shift + p' to open the vs code command palette, and start typing 'Remote-SSH:Connect' until you see the 'Remote-SSH:Connect current window to host.'
4. Select RON and enter username and password as needed. 
5. In your home directory, run```git clone https://github.com/YOURUSERNAME/prac_exam_2026.git``` with ```YOURUSERNAME``` replaced with your github username. 
6. In vscode, go to 'file --> load workspace from file', and select the 'exam-env.code-workspace' file in the 'prac_exam_2026' folder. 
7. In vscode, go to 'file --> new file', and paste your notebook/notes into the file, and save it in the 'prac_exam_2026' repo.For the practical, you should only need one vscode window open, and one terminal open.
### Pro tip: you can right click on either your notes or the exam, and select 'split right' to have both the exam and your notes open at the same time.
